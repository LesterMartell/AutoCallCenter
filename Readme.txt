AutoCallCenter v1.4

Перед началом работы, ознакомьтесь с презентацией AutoCallCenter в папке "Презентация".

Для корректной работы программы, требуется настроить открытие файлов с расширением ".ahk"

Программа подстроена под работу с Oktell - Диалоговыми формами определенного вида, а именно компании ООО "МКК" ПРОФИРЕАЛ.
Соответственно AutoCallCenter сделана именно для работы сотрудников Контакт-Центра вышеназванной компании.

Сама программа находится в архиве "AutoCallCenter v1.4.rar".
Unity - Проект можно найти по ссылке: https://drive.google.com/file/d/1BmFFlzutTsZm992fMg09E0hRcjQKQ306/view?usp=sharing

Основные Функции:
1. Выставление Городов проживания/прописки
2. Выставление стажа работы и суммы займа
3. Проверка ФИО и Даты рождения у ФССП.
4. Проверка ФИО в Microsoft Dynamics CRM на наличие действующих продуктов.
